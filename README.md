Littlewolf aims to be a very minimalistic software graphics
engine reminiscent of the early wolfenstein and doom days.

Development packages you will require:

    SDL2-devel

Then build and run with

    make; ./littlewolf

Controls:

    move: W,A,S,D

    turn: H,L

    exit: END, ESCAPE
