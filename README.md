Littlewolf aims to be a very minimalistic software graphics
engine reminiscent of the early Wolfenstein and Doom days.

    make; ./littlewolf

Dependencies:

    SDL2-devel

Controls:

    move: W,A,S,D

    turn: H,L

    exit: END, ESCAPE
